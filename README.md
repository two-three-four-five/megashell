## Getting Started

### Environment
- We tested on Mac with `Apple silicon` and `Intel chips`.

### Installation

1. Clone this repo.
``` bash
git clone https://github.com/two-three-four-five/megashell.git
```

2. Compile webserv with Makefile
```
make all
```
